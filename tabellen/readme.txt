Hier die Originalen Tabellen

File: graph.csv
This table consists on the following columns:
• Edge number
• Edge name
• Node A
• Node B
• Geographic X coordinate of node A
• Geographic Y coordinate of node A
• Geographic X coordinate of node B
• Geographic Y coordinate of node B

File: data-all.csv
This table contains the information of the different instances.
• The first row contains the edge names, and three empty columns at the end.
• Each of the subsequent row represents a different instance.
• For each instance the weights of the edges can be found in their respective column.
Additionally, the last three columns are the day of the month (1-31), the month (Mr,
Apr, or Mai), and the time stamp (hh_mm_ss), respectively.
• The edge’s weight represent the speed that is estimated to be used in the edge, as it
is measure as mph.

File: route-all.csv
This file contains the route used in each instance. Each row represents an instance,
and contains the sequence of nodes that form the used route. Also, the last three
columns are the day of the month (1-31), the month (Mr, Apr, or Mai), and the time
stamp (hh_mm_ss), respectively.
The route to be analyzed starts in node 94 and ends in node 162

Deutsch:

Datei: graph.csv
Diese Tabelle besteht aus den folgenden Spalten:
• Kantennummer
• Kantenname
• Knoten A
• Knoten B
• Geografische X-Koordinate von Knoten A
• Geografische Y-Koordinate von Knoten A
• Geografische X-Koordinate von Knoten B
• Geografische Y-Koordinate von Knoten B

Datei: data-all.csv
Diese Tabelle enthält Informationen zu den verschiedenen Instanzen.
• Die erste Zeile enthält die Kantenbezeichnungen und drei leere Spalten am Ende.
• Jede der folgenden Zeilen stellt eine andere Instanz dar.
• Für jede Instanz sind die Gewichte der Kanten in ihren jeweiligen Spalten zu finden.
Zusätzlich sind die letzten drei Spalten der Tag des Monats (1-31), der Monat (Mr, Apr oder Mai) und der Zeitstempel (hh_mm_ss).
• Das Kantengewicht repräsentiert die Geschwindigkeit, die auf der Kante geschätzt wird, da sie in mph gemessen wird.

Datei: route-all.csv
Diese Datei enthält die Route, die in jeder Instanz verwendet wird. Jede Zeile repräsentiert eine Instanz und enthält die Abfolge der Knoten, die die verwendete Route bilden. Außerdem sind die letzten drei Spalten der Tag des Monats (1-31), der Monat (Mr, Apr oder Mai) und der Zeitstempel (hh_mm_ss).
Die zu analysierende Route beginnt im Knoten 94 und endet im Knoten 162
